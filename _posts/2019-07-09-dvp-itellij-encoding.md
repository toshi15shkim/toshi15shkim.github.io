## 인텔리J UTF-8 인코딩

File > Settings > Editor > File Encodings  
UTF-8로 설정

  ![Smithsonian Image](/img/190709/1.settings.jpg){: width="700"}

<br/>

Run > Edit Configurations  
VM options : -Dfile.encoding=UTF-8 입력

  ![Smithsonian Image](/img/190709/2.Run.jpg){: width="700"}
